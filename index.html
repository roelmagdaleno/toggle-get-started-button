FAQ

How do I get my page access token?

You have to create a Facebook App first and give your Facebook Page a pages_messaging permission. Then, follow the steps 1 and 2 in the Facebook tutorial.

The generated token will NOT be saved in Facebook UI, you have to save your token in a document. Each time you select a Page from the dropdown, a new token will be generated. If a new token is generated, previously created tokens will continue to function.
The page access token is stored in this website?

No. We don't store nor share your page access token to third party.

What happen if I abuse Facebook API service?

When you click on "Save" button it will run one API request to Facebook but if you make more than 10 requests within 10 minutes, Facebook won't let you run more than that.
